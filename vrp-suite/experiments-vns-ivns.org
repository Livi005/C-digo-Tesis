* Descripción

Este archivo contiene la infraestructura de experimentación para ejecutar
VNS e IVNS sobre instancias CVRP y guardar los resultados en archivos CSV.

El objetivo del archivo no es explicar la teoría de VNS, IVNS o del CVRP,
sino organizar el código necesario para:

- cargar los módulos del sistema;
- definir una configuración experimental;
- ejecutar corridas individuales;
- ejecutar baterías completas de experimentos;
- guardar resultados resumidos en =summary.csv=;
- guardar trazas por iteración en =trace.csv=.

El archivo genera principalmente:

- =../src/experiments-vns-ivns.lisp=:
  infraestructura general de experimentación.

- =../src/ivns-experiments.lisp=:
  versión experimental de IVNS con métricas y trazas.

La documentación de este archivo está pensada para seguir el flujo del
código y entender cómo se ejecutan los experimentos desde Lisp.

* Flujo general del archivo

El flujo del archivo es el siguiente:

1. se define el paquete experimental;
2. se cargan los módulos necesarios del VRP, VNS e IVNS;
3. se define una configuración por defecto;
4. se preparan funciones auxiliares para semillas, BKS y CSV;
5. se definen funciones para ejecutar una corrida individual;
6. se exportan los resultados de cada corrida;
7. se define la ejecución masiva de experimentos;
8. se define la versión experimental de IVNS;
9. se incluyen comandos básicos para tanglear, cargar y ejecutar.

La idea es que el lector pueda seguir el archivo de arriba hacia abajo sin
tener que buscar funciones en otro lugar.

* Configuración experimental

Esta sección define la configuración base que usan las funciones de
ejecución.

La variable =*default-experiment-config*= agrupa los parámetros principales
del experimento:

- algoritmos a ejecutar;
- instancias;
- semillas;
- BKS de referencia;
- número máximo de iteraciones;
- límites de estancamiento;

(defun ensure-results-directory (path)

  "Crea la carpeta de resultados si no existe.
   Si PATH viene NIL, usa results/ por defecto."

  (let* ((safe-path
          (or path
              "results/"))

         (directory-path
          (cond
            ((pathnamep safe-path)
             safe-path)

            ((and (stringp safe-path)
                  (> (length safe-path) 0)
                  (char= (char safe-path
                               (1- (length safe-path)))
                         #\/))
             safe-path)

            ((stringp safe-path)
             (concatenate 'string
                          safe-path
                          "/"))

            (t
             "results/"))))

    (ensure-directories-exist
     (merge-pathnames
      "dummy.tmp"
      directory-path))))

(defun csv-escape-string (text)
  (with-output-to-string (out)
    (loop
      for ch across text do
        (if (char= ch #\")
            (write-string "\"\"" out)
            (write-char ch out)))))


(defun csv-safe (value)
  (cond
    ((null value)
     "")

    ((eq value t)
     "true")

    ((stringp value)
     (format nil "\"~a\""
             (csv-escape-string value)))

    ((symbolp value)
     (string-downcase
      (symbol-name value)))

    ((integerp value)
     (format nil "~d" value))

    ((numberp value)
     (format nil "~,6f"
             (coerce value 'double-float)))

    (t
     (format nil "\"~a\""
             (csv-escape-string
              (format nil "~a" value))))))

(defun write-csv-row (stream values)
  (format stream
          "~{~a~^,~}~%"
          (mapcar #'csv-safe values))

  (finish-output stream))

* Métricas experimentales registradas

La infraestructura registra métricas en dos niveles.

** Métricas por corrida

Estas métricas se guardan en =summary.csv=:

| Métrica | Significado |
|---------+-------------|
| =initial_cost= | costo de la solución inicial |
| =best_cost= | mejor costo encontrado |
| =final_cost= | costo de la solución al terminar |
| =gap_percent= | diferencia porcentual respecto al BKS |
| =iterations= | cantidad de iteraciones ejecutadas |
| =accepted_improvements= | mejoras aceptadas |
| =invalid_neighbors= | vecinos inválidos |
| =no_improvement_neighbors= | vecinos válidos que no mejoran |
| =timeout_neighbors= | vecindades detenidas por tiempo |
| =neighbors_evaluated= | cantidad de vecinos evaluados |
| =generation_tries= | intentos de generación de cadenas IVNS |
| =ivns_reset_strings_on_improvement= | indica si IVNS limpia el historial de cadenas al aceptar una mejora |
| =elapsed_time_seconds= | tiempo total de ejecución |
| =termination_reason= | razón de parada |

** Métricas por iteración

Estas métricas se guardan en =trace.csv=:

| Métrica | Significado |
|---------+-------------|
| =iter= | número de iteración |
| =k= | nivel de vecindad o contador experimental |
| =current_cost= | costo de la solución actual |
| =best_cost= | mejor costo acumulado |
| =delta= | cambio de costo |
| =accepted= | indica si la solución fue aceptada |
| =shake_applied= | indica si hubo perturbación |
| =vnd_rounds= | rondas de VND ejecutadas |
| =vnd_improvements= | mejoras dentro del VND |
| =ivns_string= | cadena generada por IVNS |
| =ivns_string_length= | longitud de la cadena IVNS |
| =ivns_complexity_score= | complejidad estimada de la cadena |
| =status= | estado de la iteración |
| =error= | mensaje de error, si existe |

Estas métricas permiten analizar calidad, estabilidad, tiempo, robustez y
comportamiento interno de los algoritmos.

* Manejo de semillas

Esta sección controla la inicialización del estado aleatorio antes de cada
corrida.

Las semillas se registran en los CSV para identificar las repeticiones
experimentales y controlar, en la medida de lo posible, la ejecución
estocástica. Sin embargo, no deben interpretarse automáticamente como un
pareo estadístico entre VNS e IVNS.

En particular, el mismo número de semilla no garantiza por sí solo que dos
algoritmos partan de una condición inicial idéntica ni que consuman la
aleatoriedad de la misma forma. IVNS puede consumir aleatoriedad adicional
durante la generación de cadenas, el filtrado, los intentos de generación y
la evaluación de vecindades.

Por esta razón, el análisis estadístico posterior debe verificar si existe
pareo real entre corridas. En ausencia de una correspondencia verificable,
las corridas se interpretan como muestras independientes bajo la misma
configuración experimental.


#+BEGIN_SRC lisp :tangle ../src/experiments-vns-ivns.lisp
(defun initialize-seed (seed)

  "Inicializa la semilla de forma reproducible.

   En SBCL se usa SB-EXT:SEED-RANDOM-STATE para que una misma
   semilla genere la misma secuencia aleatoria bajo la misma
   versión de SBCL y la misma configuración del proyecto."

  (let ((safe-seed
         (if (and seed
                  (integerp seed)
                  (>= seed 0))
             seed
             1)))

    #+sbcl
    (setf *random-state*
          (sb-ext:seed-random-state safe-seed))

    #-sbcl
    (progn
      (setf *random-state*
            (make-random-state nil))
      (loop repeat safe-seed do
            (random 1.0)))

    safe-seed))
#+END_SRC


* Mejores soluciones conocidas

Esta sección permite asociar cada instancia CVRP con su mejor solución
conocida, conocida como BKS.

El BKS se usa para calcular el gap porcentual:

- gap bajo significa que la solución encontrada está cerca de la mejor
  solución conocida;
- gap igual a cero significa que se alcanzó el BKS;
- gap positivo indica cuánto peor es la solución respecto al BKS.

Esta métrica es fundamental para comparar instancias de tamaños y costos
diferentes.

#+BEGIN_SRC lisp :tangle ../src/experiments-vns-ivns.lisp
(defun experiment-same-instance-p (a b)
  (or
   (eq a b)

   (and (symbolp a)
        (boundp a)
        (eq (symbol-value a) b))

   (and (symbolp b)
        (boundp b)
        (eq a (symbol-value b)))

   (string=
    (experiment-instance-name a)
    (experiment-instance-name b))))


(defun experiment-default-known-best (problem)
  (let ((name
         (experiment-instance-name problem)))

    (cond
      ((string= name "A-n32-k5") 784)
      ((string= name "A-n33-k5") 661)
      ((string= name "A-n33-k6") 742)
      ((string= name "A-n37-k5") 669)
      ((string= name "A-n65-k9") 1174)
      ((string= name "A-n80-k10") 1763)
      (t nil))))


(defun get-known-best (problem config)
  (or
   (loop
     for pair in (getf config :known-bests)
     when (experiment-same-instance-p problem (car pair))
       do (return (cdr pair)))

   (experiment-default-known-best problem)))
#+END_SRC


* Ejecución individual

Esta sección define cómo ejecutar una sola corrida de un algoritmo sobre una
instancia.

Las funciones principales son:

- =run-single-vns-experiment=:
  prepara los parámetros de una corrida VNS, llama a
  =vrp::vns-run-experimental= y devuelve el resultado.

- =run-single-ivns-experiment=:
  prepara los parámetros de una corrida IVNS, llama a
  =vrp::ivns-run-experimental= y devuelve el resultado.

Ambas funciones reciben información similar para que luego los resultados
puedan guardarse en el mismo formato CSV.

#+BEGIN_SRC lisp :tangle ../src/experiments-vns-ivns.lisp
(defun experiment-solution-feasible-p
       (solution
        problem
        penalty-factor)

  "Determina si una solución es factible.

   En esta infraestructura se considera factible si la evaluación
   de la solución devuelve penalización cero. Si no se puede evaluar,
   se devuelve NIL para no marcar como factible una solución dudosa."

  (if solution

      (handler-case

          (multiple-value-bind
                (cost distance penalty)

              (vrp::ivns-evaluate-solution
               solution
               problem
               penalty-factor)

            (declare
             (ignore cost distance))

            (and (numberp penalty)
                 (zerop penalty)))

        (error ()
          nil))

      nil))

(defun experiment-build-initial-solution
       (problem
        &key
          (constructor :random))

  "Construye una solución inicial común para una instancia.

   Esta solución se genera una sola vez para un par instancia-semilla
   y después se copia para cada algoritmo."
  (let* ((real-problem
          (if (symbolp problem)
              (symbol-value problem)
              problem))

         (solution
          (vrp::vrp-vns-make-initial-solution
           real-problem
           constructor)))

    solution))


(defun experiment-copy-solution (solution)

  "Copia una solución VRP para evitar compartir objetos mutables.

   VNS e IVNS deben recibir copias independientes de la misma solución
   inicial, no el mismo objeto."
  (if solution
      (vrp::clone solution)
      nil))

* Ejecución masiva

Esta sección ejecuta varias corridas automáticamente.

La función principal es =run-all-experiments=.

Esta función recorre:

- algoritmos;
- instancias;
- semillas.

Por cada combinación ejecuta una corrida individual y guarda los resultados
en los archivos CSV configurados.

La función =run-thesis-experiments= sirve como punto de entrada para lanzar
la batería definida en la configuración por defecto.

Funciones auxiliares

=experiment-build-initial-solution=: construye una solución inicial común para una instancia antes de ejecutar los algoritmos. 
Se usa cuando se desea comparar VNS e IVNS partiendo del mismo punto inicial.

=experiment-copy-solution=: crea una copia independiente de una solución inicial. 
Permite entregar la misma solución base a VNS e IVNS sin que ambos algoritmos compartan el mismo objeto mutable.

=experiment-solution-signature=: obtiene una firma textual estable de una solución inicial. 
Esta firma permite verificar en los CSV si dos corridas comenzaron desde la misma solución.

=experiment-shared-initial-solution-p=: compara las firmas iniciales reportadas por dos resultados experimentales. 
Se usa para comprobar si el pareo por solución inicial se cumplió realmente.

(defun experiment-build-initial-solution
       (problem
        &key
          (constructor :random))

  "Construye una solución inicial común para experimentos pareados.

   Esta función se usa cuando se desea que VNS e IVNS partan de la misma
   solución inicial para una misma instancia y una misma semilla."
  (let* ((real-problem
          (if (symbolp problem)
              (symbol-value problem)
              problem))

         (solution
          (cond
            ((or (eq constructor :random)
                 (null constructor))
             (vrp::vrp-vns-make-initial-solution
              real-problem
              :random))

            ((functionp constructor)
             (funcall constructor
                      real-problem))

            (t
             (vrp::vrp-vns-make-initial-solution
              real-problem
              constructor)))))

    solution))


(defun experiment-copy-solution (solution)

  "Copia una solución VRP.

   Se usa para que VNS e IVNS reciban copias independientes de la misma
   solución inicial. Esto evita que un algoritmo modifique el mismo objeto
   que luego utilizaría el otro."
  (if solution
      (vrp::clone solution)
      nil))

   La firma permite comprobar si dos corridas comenzaron desde la misma
   solución. Si la solución es NIL, devuelve una cadena vacía."
  (if solution

      ""))


(defun experiment-shared-initial-solution-p
       (vns-result
        ivns-result)

  "Comprueba si dos resultados reportan la misma firma inicial.

   Devuelve T si las firmas iniciales de VNS e IVNS coinciden."
  (let ((vns-signature
         (getf vns-result :initial-solution-signature))

        (ivns-signature
         (getf ivns-result :initial-solution-signature)))

    (and vns-signature
         ivns-signature
         (string= vns-signature
                  ivns-signature))))

* Núcleo experimental de IVNS

Esta sección define la versión experimental de IVNS.

El bloque tangla hacia:

- =../src/ivns-experiments.lisp=

Esta versión no solo ejecuta IVNS, sino que también registra métricas
compatibles con la infraestructura experimental.

El flujo básico de una iteración IVNS es:

1. generar una cadena IVNS;
2. validar la cadena;
3. traducirla a un criterio VRP;
4. aplicar el criterio sobre la solución actual;
5. evaluar el vecino generado;
6. aplicar VND si está habilitado;
7. decidir si se acepta la nueva solución;
8. guardar una fila de traza.

Esta sección permite que IVNS produzca resultados comparables con VNS en
=summary.csv= y =trace.csv=.

#+BEGIN_SRC lisp :tangle ../src/ivns-experiments.lisp
(in-package :vrp)

(defun experiment-set-random-seed (seed)

  "Inicializa *random-state* de forma reproducible para cada corrida.

   Esta función debe usarse al inicio de cada corrida experimental.
   El objetivo es que una misma semilla produzca la misma corrida
   bajo la misma versión de SBCL y la misma configuración del proyecto."

  (let ((safe-seed
         (if (and seed
                  (integerp seed)
                  (>= seed 0))
             seed
             1)))

    #+sbcl
    (setf *random-state*
          (sb-ext:seed-random-state safe-seed))

    #-sbcl
    (progn
      (setf *random-state*
            (make-random-state nil))
      (loop repeat safe-seed do
            (random 1.0)))

    safe-seed))


(defun experiment-solution-feasible-p
       (solution problem &optional (penalty-factor 1000))

  "Determina si una solución se considera factible.

   En esta versión experimental, una solución es factible si la
   evaluación del framework devuelve penalización cero.

   Esto mantiene la factibilidad alineada con la función objetivo
   usada durante las corridas."

  (handler-case

      (multiple-value-bind
            (cost distance penalty)

          (ivns-evaluate-solution
           solution
           problem
           penalty-factor)

        (declare
         (ignore cost distance))

        (and (numberp penalty)
             (zerop penalty)))

    (error ()
      nil)))


(defun experiment-safe-number (value &optional (default 0))

  "Devuelve VALUE si es numérico; en otro caso devuelve DEFAULT."

  (if (numberp value)
      value
      default))


(defun experiment-safe-string (value &optional (default ""))

  "Devuelve VALUE como string seguro para exportación."

  (cond
    ((stringp value)
     value)

    ((symbolp value)

      (symbol-name value)))

    ((null value)

(defun ivns-generate-new-string
       (used-strings
        max-tries)

  "Genera una cadena IVNS y devuelve diagnóstico.

   Devuelve cuatro valores, en el orden que espera IVNS-RUN-EXPERIMENTAL:

   1. cadena generada, o NIL si no se pudo generar;
   2. cantidad de intentos realizados;
   3. cantidad de cadenas rechazadas;
   4. plist diagnóstico.

   Esta función no interpreta USED-STRINGS como parámetro de configuración.
   USED-STRINGS representa las cadenas ya usadas dentro de la corrida IVNS
   actual. Si el filtro genera una cadena repetida dentro de la misma corrida,
   se rechaza y se intenta generar otra."

  (let ((cadena nil)
        (tries 0)
        (rejected 0)
        (generated-count-total 0)
        (valid-count-total 0)
        (invalid-count-total 0)
        (last-reason "")
        (last-key :unknown)
        (last-reject-counts nil))

    (labels
        ((safe-reason-key
             (reason)

           (handler-case
               (ivns::ivns-reject-reason->key reason)

             (error ()
               :unknown)))

         (make-diagnostic
             ()

           (list
            :string cadena

            :generated-count generated-count-total
            :valid-count valid-count-total
            :invalid-count invalid-count-total

            :reject-reason last-reason
            :reject-key last-key
            :reject-counts last-reject-counts

            :generation-tries tries
            :generation-max-tries max-tries

            :strings-generated generated-count-total
            :strings-valid valid-count-total
            :strings-invalid invalid-count-total)))

      (loop while (and (< tries max-tries)
                       (null cadena))

            do
              (multiple-value-bind
                    (generated-string attempts invalid-count-local
                     last-reason-local reason-counts
                     generated-count-local valid-count-local)

                  (ivns::filter-string-with-diagnostics
                   :max-attempts max-tries
                   :max-imbalance
                   (ivns::get-config :max-imbalance 2)
                   :max-length
                   (ivns::get-config :max-length 10)
                   :max-complexity
                   (ivns::get-config :max-complexity 14)
                   :allow-expensive-patterns
                   (ivns::get-config :allow-expensive-patterns nil))

                (incf tries
                      (max 1
                           (or attempts
                               1)))

                (incf generated-count-total
                      (or generated-count-local
                          0))

                (incf valid-count-total
                      (or valid-count-local
                          0))

                (incf invalid-count-total
                      (or invalid-count-local
                          0))

                (setf last-reason
                      (or last-reason-local
                          ""))

                (setf last-key
                      (safe-reason-key
                       last-reason))

                (setf last-reject-counts
                      reason-counts)

                (cond

                  ((null generated-string)
                   (incf rejected)

                   (when (or (null last-reason)
                             (= (length last-reason)
                                0))
                     (setf last-reason
                           "generation-limit-reached")

                     (setf last-key
                           :generation-limit-reached)))

                  ((member generated-string
                           used-strings
                           :test #'equal)

                   (incf rejected)
                   (incf invalid-count-total)

                   (setf last-reason
                         "duplicated-string")

                   (setf last-key
                         :duplicated-string)

                   (let ((current
                          (getf last-reject-counts
                                :duplicated-string
                                0)))

                     (setf (getf last-reject-counts
                                 :duplicated-string)
                           (1+ current))))

                  (t
                   (setf cadena
                         generated-string)))))

      (when (and (null cadena)
                 (or (null last-reason)
                     (= (length last-reason)
                        0)))

        (setf last-reason
              "generation-limit-reached")

        (setf last-key
              :generation-limit-reached))

      (values
       cadena
       tries
       rejected
       (make-diagnostic)))))

           solution
           problem
           action
           max-neighborhood-seconds)))

    (error (e)

      (values
       nil
       :error
       (format nil "~a" e)))))

* Notas

** =k= y =kmax= en IVNS

En VNS clásico, =k= indica el nivel de vecindad utilizado por Shake.

En IVNS, la vecindad no se toma de una lista fija indexada por =k=. La
variación se produce mediante la generación de nuevas cadenas gramaticales.

Por esa razón, en esta versión experimental =k= se conserva como contador
homologado para trazas y control de estancamiento, pero la generación de
vecindades depende principalmente de:

- =generation-max-tries=;
- =ivns-max-neighborhood-seconds=;
- filtros de cadenas IVNS;
- gramática usada por el generador.

Esta aclaración evita interpretar incorrectamente el significado de =k= al
comparar VNS e IVNS.

** Reinicio de cadenas usadas al mejorar

El parámetro =reset-generated-strings-on-improvement-p= controla si IVNS
limpia el historial de cadenas ya usadas cuando acepta una mejora.

- Si es =T=, después de una mejora se reinicia =k= y también se limpia el
  historial local de cadenas.
- Si es =NIL=, después de una mejora se reinicia =k=, pero las cadenas usadas
  durante la corrida se conservan.

Este parámetro permite comparar ambas políticas sin cambiar el código interno
de la corrida.

** =shake-max-tries= en IVNS

El parámetro =shake-max-tries= se mantiene para que VNS e IVNS compartan una
estructura experimental común.

En VNS, este parámetro controla los intentos del Shake clásico.

En IVNS, la perturbación se basa en la generación de cadenas gramaticales.
Por tanto, el control efectivo del Shake IVNS depende sobre todo de
=generation-max-tries= y =ivns-max-neighborhood-seconds=.

Si se desea una equivalencia más estricta, puede definirse
=shake-max-tries= como alias de =generation-max-tries= o eliminarse de la
firma de IVNS experimental. Mientras se mantenga, debe quedar documentado.

            (if generated-string

                   :strings-generated generated-count

       (candidates)

  "Calcula estadísticas de una lista de candidatas IVNS.

   - candidatas generadas;
   - cadenas generadas por el filtro;
   - cadenas válidas;

                   (getf counts :unbalanced 0))

                   (getf counts :missing-required-symbol 0))

             (incf reject-forbidden-expensive-pattern
                   (getf counts :forbidden-expensive-pattern 0))

             (incf reject-semantic-invalid
                   (getf counts :semantic-invalid 0))

             (incf reject-duplicated-string
                   (getf counts :duplicated-string 0))

             (incf reject-generation-limit-reached
                   (getf counts :generation-limit-reached 0))

             (incf reject-unknown
                   (getf counts :unknown 0)))))

      (dolist (candidate candidates)

              (or (getf candidate :strings-invalid)
                  0))

    (list

     :reject-missing-required-symbol reject-missing-required-symbol

     :reject-semantic-invalid reject-semantic-invalid

(defun ivns-best-candidate (valid-candidates)
  "Devuelve la candidata válida con menor delta.
   Puede ser mejorante o no mejorante; si todas empeoran,
   devuelve la que menos empeora."
  (first
   (sort (copy-list valid-candidates)
         #'<
         :key (lambda (candidate)
                (getf candidate :delta)))))

(defun ivns-select-candidate
       (candidates selection-mode exploration-rate)
  "Selecciona una candidata según SELECTION-MODE.

   Modos:
   :single
   :first-valid
   :best
   :mixed

   Devuelve:
   selected-candidate
   selected-reason"
  (let* ((mode (or selection-mode :single))
         (valid-candidates
          (remove-if-not #'ivns-candidate-valid-p candidates))
         (improving-candidates
          (remove-if-not #'ivns-candidate-improving-p valid-candidates)))

    (cond
      ;; Comportamiento equivalente al IVNS básico:
      ;; una sola cadena por iteración.
      ((eq mode :single)
       (let ((candidate (first valid-candidates)))
         (values candidate
                 (if candidate
                     "single"
                     "none_valid"))))

      ;; Usa la primera cadena válida.
      ((eq mode :first-valid)
       (let ((candidate (first valid-candidates)))
         (values candidate
                 (if candidate
                     "first_valid"
                     "none_valid"))))

      ;; Usa la mejor candidata por delta.
      ((eq mode :best)
       (let ((candidate (ivns-best-candidate valid-candidates)))
         (values candidate
                 (cond
                   ((null candidate) "none_valid")
                   ((member candidate improving-candidates) "best_improving")
                   (t "best_valid")))))

      ;; Mezcla explotación y exploración.
      ;; Con probabilidad exploration-rate escoge una válida aleatoria.
      ;; En caso contrario escoge la mejor por delta.
      ((eq mode :mixed)
       (let ((candidate
              (if (and valid-candidates
                       (< (random 1.0) exploration-rate))
                  (ivns-random-elt valid-candidates)
                  (ivns-best-candidate valid-candidates))))
         (values candidate
                 (cond
                   ((null candidate) "none_valid")
                   ((member candidate improving-candidates)
                    "mixed_best_or_random_improving")
                   (t
                    "mixed_random_or_best_valid")))))

      ;; Si llega un modo desconocido, usa :single.
      (t
       (let ((candidate (first valid-candidates)))
         (values candidate
                 (if candidate
                     "single_fallback"
                     "none_valid")))))))

(defun ivns-generate-and-select-shake-candidate
       (current
        problem
        current-cost
        penalty-factor
        max-neighborhood-seconds
        candidates-per-iteration
        selection-mode
        exploration-rate
        generation-max-tries)

  "Genera varias candidatas IVNS y selecciona una.

   GENERATION-MAX-TRIES se pasa a IVNS-BUILD-SHAKE-CANDIDATE para que
   el parámetro experimental controle realmente el filtrado de cadenas."

  (let ((candidates '()))

    (loop for i from 1 to (max 1 candidates-per-iteration)

              :generation-max-tries generation-max-tries)

             (ivns-candidate-stats candidates)))

          (setf (getf stats :strings-selected-total)
                1)

          (when (ivns-candidate-improving-p selected)

            (setf (getf stats :strings-accepted-total)
                  1)))

        (values

         stats
         reason
         candidates)))))

(defun ivns-vnd-search
       (solution problem criteria penalty-factor
        vnd-mode vnd-max-rounds vnd-max-no-improve)

  "Aplica búsqueda local sobre la solución producida por el shake IVNS.

   Modos soportados:

   - :best selecciona la mejor mejora encontrada.
   - :first selecciona la primera mejora encontrada.
   - :random selecciona aleatoriamente una mejora válida.

   El modo :random no acepta cualquier vecino, sino que escoge al azar
   entre vecinos que mejoran el costo actual."
  (let* ((current
          solution)

         (best
          solution)

         (best-cost
          (multiple-value-bind
                (cost distance penalty)

              (ivns-evaluate-solution
               solution
               problem
               penalty-factor)

            (declare
             (ignore distance penalty))

            cost))

         (rounds 0)
         (improvements 0)
         (neighbors-evaluated 0)
         (no-improve 0))

    (labels
        ((solution-cost (candidate)

           (multiple-value-bind
                 (cost distance penalty)

               (ivns-evaluate-solution
                candidate
                problem
                penalty-factor)

             (declare
              (ignore distance penalty))

             cost))

         (select-random-improvement (candidates)

           (when candidates
             (nth (random (length candidates))
                  candidates))))

      (if (or (null criteria)
              (null vnd-max-rounds)
              (<= vnd-max-rounds 0)
              (eq vnd-mode :none))

          (values
           best
           best-cost
           0
           0
           0)

          (progn

            (loop while
                  (and (< rounds vnd-max-rounds)
                       (< no-improve vnd-max-no-improve))

                  do
                    (incf rounds)

                    (let ((round-improved nil)
                          (round-best-solution nil)
                          (round-best-cost best-cost)
                          (random-improving-candidates '()))

                      ;; Recorre los criterios de búsqueda local.
                      ;; El CATCH permite salir de la ronda cuando el modo
                      ;; es :first y aparece la primera mejora.
                      (catch 'finish-round

                        (dolist (criterion criteria)

                          (handler-case

                              (let ((neighbor
                                     (funcall criterion
                                              current
                                              problem
                                              nil)))

                                (incf neighbors-evaluated)

                                (when neighbor

                                  (let ((neighbor-cost
                                         (solution-cost neighbor)))

                                    (when (< neighbor-cost best-cost)

                                      (cond

                                        ;; Primera mejora.
                                        ((eq vnd-mode :first)

                                         (setf best neighbor)
                                         (setf best-cost neighbor-cost)
                                         (setf current neighbor)

                                         (setf round-improved t)
                                         (incf improvements)

                                         (throw 'finish-round t))

                                        ;; Mejora aleatoria.
                                        ((eq vnd-mode :random)

                                         (push
                                          (list
                                           :solution neighbor
                                           :cost neighbor-cost)
                                          random-improving-candidates))

                                        ;; Mejor mejora.
                                        (t

                                         (when (< neighbor-cost
                                                  round-best-cost)

                                           (setf round-best-solution
                                                 neighbor)

                                           (setf round-best-cost
                                                 neighbor-cost)))))))))

                            (error ()
                              nil)))

                      ;; Si el modo es :random, seleccionar aleatoriamente
                      ;; una mejora válida entre las encontradas.
                      (when (and (eq vnd-mode :random)
                                 random-improving-candidates)

                        (let* ((selected
                                (select-random-improvement
                                 random-improving-candidates))

                               (selected-solution
                                (getf selected :solution))

                               (selected-cost
                                (getf selected :cost)))

                          (setf best selected-solution)
                          (setf best-cost selected-cost)
                          (setf current selected-solution)

                          (setf round-improved t)
                          (incf improvements)))

                      ;; Si el modo es :best, aplicar la mejor mejora
                      ;; encontrada en la ronda.
                      (when (and (not (member vnd-mode
                                              '(:first :random :none)))
                                 round-best-solution)

                        (setf best round-best-solution)
                        (setf best-cost round-best-cost)
                        (setf current round-best-solution)

                        (setf round-improved t)
                        (incf improvements))

                      (if round-improved
                          (setf no-improve 0)
                          (incf no-improve))))

            (values
             best
             best-cost
             rounds
             improvements
             neighbors-evaluated))))))

(defun ivns-run-experimental
       (problem
        &key
          criteria
          (constructor #'make-initial-solution-for-cvrp-random)
          (initial-solution nil)
          (initial-solution-source "generated_inside_algorithm")
          (initial-solution-shared-p nil)
          (seed 1)
          (max-iter 100)
          (kmax 5)
          (max-no-improve 30)
          (shake-max-tries 10)
          (generation-max-tries 50)
          (vnd-mode :best-improvement)
          (vnd-max-rounds 20)
          (vnd-max-no-improve 10)
          (ivns-max-neighborhood-seconds 0.05)
          (penalty-factor 1000)
          (reset-generated-strings-on-improvement-p t)
          known-best
          time-limit-seconds
          quiet
          verbose)

  (declare (ignore criteria verbose))

  (setf *random-state*
        (make-random-state t))

  (loop repeat seed do
        (random 1.0))
 
  (when (fboundp 'ivns::clear-generated-strings)
        (ivns::clear-generated-strings))

  (let* ((initial-solution-real
          (if initial-solution
              (vrp::clone initial-solution)
              (funcall constructor problem)))

         (initial-cost
          (multiple-value-bind
                (cost distance penalty)

              (ivns-evaluate-solution
               initial-solution-real
               problem
               penalty-factor)

            (declare (ignore distance penalty))
            cost))

         (initial-solution-signature
          (ivns-solution-signature initial-solution-real))

         (current initial-solution-real)
         (current-cost initial-cost)

         (best (vrp::clone current))
         (best-cost current-cost)
         (final-cost current-cost)

         (iter 0)
         (k 1)
         (no-improve 0)

         (accepted-improvements 0)

         (invalid-neighbors 0)
         (no-improvement-neighbors 0)
         (timeout-neighbors 0)
         (neighbors-evaluated-total 0)

         (generation-tries-total 0)

         (strings-generated-total 0)
         (strings-valid-total 0)
         (strings-invalid-total 0)
         (strings-evaluated-total 0)
         (strings-selected-total 0)
         (strings-accepted-total 0)

         (reject-too-many-r 0)
         (reject-too-short 0)
         (reject-too-long 0)
         (reject-unbalanced 0)
         (reject-missing-required-symbol 0)
         (reject-forbidden-expensive-pattern 0)
         (reject-semantic-invalid 0)
         (reject-duplicated-string 0)
         (reject-generation-limit-reached 0)
         (reject-unknown 0)

         (unique-strings
          (make-hash-table :test 'equal))

         (used-strings '())

         (best-iteration 0)
         (termination-reason "unknown")
         (trace '())

         (start-time
          (get-internal-real-time)))

    (labels
        ((elapsed-seconds ()
           (/ (- (get-internal-real-time)
                 start-time)
              internal-time-units-per-second))

         (safe-next-k ()
           (incf k)
           (when (> k kmax)
             (setf k 1)))

         (register-reject-key (key)
           (case key
             (:too-many-r
              (incf reject-too-many-r))

             (:too-short
              (incf reject-too-short))

             (:too-long
              (incf reject-too-long))

             (:unbalanced
              (incf reject-unbalanced))

             (:missing-required-symbol
              (incf reject-missing-required-symbol))

             (:forbidden-expensive-pattern
              (incf reject-forbidden-expensive-pattern))

             (:semantic-invalid
              (incf reject-semantic-invalid))

             (:duplicated-string
              (incf reject-duplicated-string))

             (:generation-limit-reached
              (incf reject-generation-limit-reached))

             (otherwise
              (incf reject-unknown)))))

      (loop
        while (and (< iter max-iter)
                   (< no-improve max-no-improve)
                   (or (null time-limit-seconds)
                       (< (elapsed-seconds)
                          time-limit-seconds)))

        do
          (incf iter)

          (let* ((generated-string nil)
                 (complexity-score 0)

                 (generation-tries 0)
                 (generation-rejected 0)

                 (filter-reject-reason "")
                 (filter-reject-key :ok)

                 (strings-generated-iter 0)
                 (strings-valid-iter 0)
                 (strings-invalid-iter 0)
                 (strings-evaluated-iter 0)
                 (strings-selected-iter 0)
                 (strings-accepted-iter 0)

                 (neighbor nil)
                 (neighbor-cost nil)
                 (distance 0)
                 (penalty 0)
                 (delta 0)
                 (accepted nil)

                 (row-status "ok")
                 (row-error ""))

            ;; Generación de cadena con diagnóstico.
            (multiple-value-bind
                  (cadena tries rejected diagnostic)

                (ivns-generate-new-string
                 used-strings
                 generation-max-tries)

              (setf generated-string cadena)
              (setf generation-tries tries)
              (setf generation-rejected rejected)

              (setf strings-generated-iter
                    (or (getf diagnostic :generated-count)
                        tries))

              (setf strings-valid-iter
                    (or (getf diagnostic :valid-count)
                        0))

              (setf strings-invalid-iter
                    (or (getf diagnostic :invalid-count)
                        rejected))

              (setf filter-reject-reason
                    (or (getf diagnostic :reject-reason)
                        ""))

              (setf filter-reject-key
                    (or (getf diagnostic :reject-key)
                        :unknown))

              (incf generation-tries-total tries)
              (incf strings-generated-total strings-generated-iter)
              (incf strings-valid-total strings-valid-iter)
              (incf strings-invalid-total strings-invalid-iter)

              (when generated-string
                (push generated-string used-strings)
                (setf (gethash generated-string unique-strings) t)
                (setf complexity-score
                      (ivns::ivns-complexity-score
                       generated-string))))

            ;; Si no hay cadena válida, se registra como fallo de generación.
            (unless generated-string
              (setf row-status "none_valid")
              (setf row-error filter-reject-reason)
              (register-reject-key filter-reject-key)
              (incf invalid-neighbors)
              (incf no-improve)
              (safe-next-k))

            ;; Convertir cadena en criterio y aplicarla.
            (when generated-string
              (handler-case
                  (multiple-value-bind
                        (res actions tokens)

                      (ivns::apply-subindices
                       generated-string)

                    (declare (ignore res tokens))

                    (let* ((criterion
                            (ivns-actions->criterion
                             actions))

                           (action
                            (delta-cvrp-action)))

                      (incf strings-selected-iter)
                      (incf strings-selected-total)

                      (multiple-value-bind
                            (candidate status error-message)

                          (ivns-safe-apply-criterion
                           criterion
                           current
                           problem
                           action
                           ivns-max-neighborhood-seconds)

                        (setf neighbor candidate)

                        (cond
                          ((eq status :timeout)
                           (setf row-status "timeout")
                           (setf row-error error-message)
                           (incf timeout-neighbors))

                          ((eq status :error)
                           (setf row-status "error")
                           (setf row-error error-message)
                           (incf invalid-neighbors))

                          (t
                           (setf row-status "ok")
                           (setf row-error ""))))))

                (error (e)
                  (setf row-status "error")
                  (setf row-error
                        (format nil "~a" e))
                  (incf invalid-neighbors))))

            ;; Evaluación del vecino.
            (when neighbor
              (incf strings-evaluated-iter)
              (incf strings-evaluated-total)
              (incf neighbors-evaluated-total)

              (multiple-value-bind
                    (cost dist pen)

                  (ivns-evaluate-solution
                   neighbor
                   problem
                   penalty-factor)

                (setf neighbor-cost cost)
                (setf distance dist)
                (setf penalty pen)

                (setf delta
                      (- neighbor-cost
                         current-cost))

                (if (< neighbor-cost
                       current-cost)

                    (progn
                      (setf accepted t)
                      (incf accepted-improvements)
                      (incf strings-accepted-iter)
                      (incf strings-accepted-total)

                      (setf current neighbor)
                      (setf current-cost neighbor-cost)
                      (setf final-cost neighbor-cost)

                      (setf no-improve 0)
                      (setf k 1)
                      

                      (when reset-generated-strings-on-improvement-p
                        (when (fboundp 'ivns::clear-generated-strings)
                          (ivns::clear-generated-strings))
                        (setf used-strings '()))

                      (when (< current-cost
                               best-cost)
                        (setf best
                              (vrp::clone current))
                        (setf best-cost current-cost)
                        (setf best-iteration iter)))

                    (progn
                      (incf no-improvement-neighbors)
                      (incf no-improve)
                      (safe-next-k)))))

            ;; Si hubo cadena pero no vecino, cuenta como intento no aplicable.
            (when (and generated-string
                       (null neighbor)
                       (string= row-status "ok"))
              (setf row-status "none_valid")
              (setf row-error "La cadena no produjo vecino aplicable.")
              (incf invalid-neighbors)
              (incf no-improve)
              (safe-next-k))

            ;; Trace.
            (push
             (ivns-make-trace-row
              :iter iter
              :k k
              :cadena generated-string
              :generation-tries generation-tries
              :rejected generation-rejected
              :accepted accepted
              :nil-neighbor (null neighbor)
              :current-cost current-cost
              :best-cost best-cost
              :delta delta
              :distance distance
              :penalty penalty
              :elapsed-time (elapsed-seconds)
              :status row-status
              :error row-error
              :strings-generated-iter strings-generated-iter
              :strings-valid-iter strings-valid-iter
              :strings-invalid-iter strings-invalid-iter
              :strings-evaluated-iter strings-evaluated-iter
              :strings-selected-iter strings-selected-iter
              :strings-accepted-iter strings-accepted-iter
              :unique-strings-iter (hash-table-count unique-strings)
              :filter-reject-reason filter-reject-reason
              :filter-reject-key filter-reject-key)
             trace)

            (when (not quiet)
              (format t
                      "~%[ITER ~3d] cost=~8,2f | best=~8,2f | delta=~8,2f | accepted=~a | k=~d | string=~a | status=~a"
                      iter
                      current-cost
                      best-cost
                      delta
                      accepted
                      k
                      generated-string
                      row-status))))

      (cond
        ((>= iter max-iter)
         (setf termination-reason "max_iter"))

        ((>= no-improve max-no-improve)
         (setf termination-reason "max_no_improve"))

        ((and time-limit-seconds
              (>= (elapsed-seconds)
                  time-limit-seconds))
         (setf termination-reason "time_limit"))

        (t
         (setf termination-reason "completed")))

      (list
       :algorithm "ivns"

       :known-best known-best

       :max-iter max-iter
       :kmax kmax
       :max-no-improve max-no-improve

       :shake-max-tries shake-max-tries

       :generation-max-tries generation-max-tries

       :vnd-mode vnd-mode
       :vnd-max-rounds vnd-max-rounds
       :vnd-max-no-improve vnd-max-no-improve

       :time-limit-seconds time-limit-seconds
       :ivns-max-neighborhood-seconds ivns-max-neighborhood-seconds
       :reset-generated-strings-on-improvement-p reset-generated-strings-on-improvement-p

       :termination-reason termination-reason

       :initial-cost initial-cost
       :initial-solution-signature initial-solution-signature
       :initial-solution-shared-p initial-solution-shared-p
       :initial-solution-source initial-solution-source

       :best-cost best-cost
       :final-cost final-cost

       :gap
       (if known-best
           (* 100.0
              (/ (- best-cost known-best)
                 known-best))
           0.0)

       :iterations iter

       :accepted-improvements accepted-improvements

       :invalid-neighbors invalid-neighbors
       :no-improvement-neighbors no-improvement-neighbors
       :timeout-neighbors timeout-neighbors

       :generation-tries generation-tries-total
       :neighbors-evaluated neighbors-evaluated-total

       :best-iteration best-iteration
       :stagnation-iterations no-improve

       :time (elapsed-seconds)
       :elapsed-time-seconds (elapsed-seconds)

       :feasible t
       :status "ok"
       :error ""

       :ivns-strings-generated-total strings-generated-total
       :ivns-strings-valid-total strings-valid-total
       :ivns-strings-invalid-total strings-invalid-total
       :ivns-strings-evaluated-total strings-evaluated-total
       :ivns-strings-selected-total strings-selected-total
       :ivns-strings-accepted-total strings-accepted-total
       :ivns-unique-strings-total (hash-table-count unique-strings)

       :ivns-reject-too-many-r reject-too-many-r
       :ivns-reject-too-short reject-too-short
       :ivns-reject-too-long reject-too-long
       :ivns-reject-unbalanced reject-unbalanced
       :ivns-reject-missing-required-symbol reject-missing-required-symbol
       :ivns-reject-forbidden-expensive-pattern reject-forbidden-expensive-pattern
       :ivns-reject-semantic-invalid reject-semantic-invalid
       :ivns-reject-duplicated-string reject-duplicated-string
       :ivns-reject-generation-limit-reached reject-generation-limit-reached
       :ivns-reject-unknown reject-unknown

       :trace (reverse trace)
       :best-solution best))))

** Experimento principal VNS vs IVNS

El experimento principal compara VNS e IVNS con el perfil =baseline= y tiempo
de vecindad IVNS =0.05=. Se evalúan tres instancias, tres modos de búsqueda
local y treinta semillas.

Diseño:

- algoritmos: VNS e IVNS;
- perfil IVNS: =baseline=;
- tiempo de vecindad IVNS: =0.05=;
- reinicio de cadenas IVNS al mejorar: activado por defecto;
- modos: =best=, =first=, =random=;
- instancias: =A-n33-k5=, =A-n65-k9=, =A-n80-k10=;
- semillas: 1 a 30;
- total esperado: 540 filas =summary=.

Crear el script:

cat > commands/10-main-vns-ivns-baseline.sh <<'BASH'
#!/usr/bin/env bash
set -euo pipefail

INSTANCES=(
"a-n33-k5 vrp::a-n33-k5-problem 45 30m"
"a-n65-k9 vrp::a-n65-k9-problem 60 45m"
"a-n80-k10 vrp::a-n80-k10-problem 75 60m"
)

BLOCKS=(
"s01-05 (list 1 2 3 4 5)"
"s06-10 (list 6 7 8 9 10)"
"s11-15 (list 11 12 13 14 15)"
"s16-20 (list 16 17 18 19 20)"
"s21-25 (list 21 22 23 24 25)"
"s26-30 (list 26 27 28 29 30)"
)

MODES=(
"best"
"first"
"random"
)

for mode in "${MODES[@]}"; do
for inst_item in "${INSTANCES[@]}"; do
read -r INST_LABEL INST_SYMBOL INTERNAL_LIMIT EXTERNAL_TIMEOUT <<< "$inst_item"

for block_item in "${BLOCKS[@]}"; do
  BLOCK_LABEL="${block_item%% *}"
  SEEDS_EXPR="${block_item#* }"

  ./run_direct_block.sh \
    main-${mode}-baseline-0p05-${INST_LABEL}-${BLOCK_LABEL} \
    ${INST_SYMBOL} \
    "${SEEDS_EXPR}" \
    "(list :vns :ivns)" \
    baseline \
    ${mode} \
    0.05 \
    t \
    ${INTERNAL_LIMIT} \
    50 \
    ${EXTERNAL_TIMEOUT}
done

done
done
